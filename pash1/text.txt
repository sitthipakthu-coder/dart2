โครงสร้างคำสั่งภาษา Dart
main() คือ ฟังก์ชั่นพิเศษกลุ่มคำสั่งที่อยู่ในฟังก์ชั่นนี้ จะทำงานโดยอัตโนมัติในตอนเริ่มต้นเสมอ

ขอบเขต(Block) ใช้สัญลักษณ์ {} เพื่อกำหนดขอบเขตการทำงานของกลุ่มคำสั่งนั้น

เครื่องหมายสิ้นสุดคำสั่ง ;
คำอธิบาย(Comment) // หรือ /**/

การสร้างตัวแปร มี 2 แบบ
Static Type สร้างแบบระบุชนิดข้อมูล
Dynamic Type สร้างแบบไม่ระบุชนิดข้อมูล

ชนิดข้อมูล
int  ข้อมูลเต็มบวกและลบ
double ตัวเลขทศนิมยม
num ตัวรวมของ int และ double
bool ข้อมูลทางตรรกศาสตร์ จริงและเท็จ
String ข้อมูลข้อความ
Dynamic ข้อมูลชนิดอะไรก็ได้
var  อ้างอิงตามข้อมูลที่ใส่

ตัวแปรแบบ Dynamic
ใช้กับกรณีตัวแปรไม่แน่นอน โดยที่ตัวแปรแบบนี้จะเก็บข้อมูลชนิดใดก็ได้

ค่าคงที่ นิยมใช้เป็นแบบตัวพิมพ์ใหญ่ทั้งหมด
สร้างแบบ const ต้องระบุชนิดข้อมูล final ไม่ต้องระบุชนิดข้อมูลอ้างอิงตามข้อมูลที่ใส่มา

การสร้างด้วย const ต้องกำหนดค่าเริ่มต้นในตอนประกาศใช้งานเสมอและแก้ไขภายหลังไม่ได้
การสร้างด้วย final สามารถประกาศไว้ก่อนและกำหนดภายหลังได้ แต่จะกำหนดได้ครั้งเดียว

ตัวดำเนินการ (Operators)
1.ตัวดำเนินการ (Operator)
2.ตัวถูกดำเนินการ (Operand)

ตัวดำนเนินการเพิ่มและลดค่า
++(Prefix) ++a เพื่อค่าให้ a ก่อน แล้วค่อยนำไปใช้
++(Postfix) a++ เอาไปใช้ก่อนแล้วค่อยเพิ่มค่า
--(Prefix) --a ลบค่าให้ a ก่อน แล้วค่อยนำไปใช้
--(Postfix) a-- เอาไปใช้ก่อนแล้วค่อยลดค่า

ตัวดำนเนิการกำหนดค่า
+= x+=y = x=x+y
-= x-=y = x=x-y
*= x*=y = x=x*y
/= x/=y = x=x/y
%= x%=y = x=x%y
~/= x~/y = x=x~/y

== เท่ากันกับ
!= ไม่เท่ากับ
> มากกว่า
> น้อยกว่า
>= มากกว่าเท่ากับ
<= น้อยกว่าเท่ากับ

ตัวดำเนินการทางตรรกศาสตร์
&& AND และ
|| OR หรือ
! NOT ไม่

Switch .. CaseSwitch เป็นคำสั่งที่ใช้กำหนดเงื่อนไขคล้ายๆกับ if แต่จะเลือกเพียงหนึ่งทางเลือกออกมาทำงานโดยนำค่าในตัวแปรมากำหนดเป็นทางเลือกผ่านคำสั่งcase (ค่าที่ต้องการตรวจสอบ)

break คือ คำสั่งนี้จะหลุดการทำงานจากในลูปทันที เพื่อไปทำคำสั่งอื่น

continue คำสั่งนี้จะทำให้หยุดการทำงานแล้วย้อนเข้าลูปใหม่อีกครั้ง

ข้อแตกต่างการใช้รูป Loop
For ใช้กับจำนวนรอบที่ชัดเจน
While ใช้ในกรณีไม่รู้จำนวนรอบ
Do..While ใช้กรณีอยากลดลองการ 1 รอบ แล้วทำซ้ำตามเงื่อนไข
